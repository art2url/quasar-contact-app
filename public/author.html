I'm Artem Turlenko, a passionate full-stack developer with a strong focus on privacy, security, and creating meaningful digital experiences. I created Quasar Contact because I believe that private communication should be accessible to everyone, without compromising on security or usability.

My Background

With extensive experience in modern web technologies including Angular, Node.js, and real-time communication systems, I've dedicated my career to building secure, user-friendly applications. My expertise spans frontend development, backend architecture, encryption implementation, and database design.

Why Quasar Contact?

In today's digital landscape, privacy is becoming increasingly rare. I built Quasar Contact to address this challenge - creating a messaging platform that puts user privacy first, implements true end-to-end encryption, and maintains a zero-knowledge architecture. Every technical decision is made with user privacy and security in mind.

Philosophy

I believe that:

  • Privacy is a fundamental right - Not a luxury or optional feature
  • Security should be transparent - Users should understand how their data is protected
